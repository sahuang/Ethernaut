# Ethernaut

My walkthrough and learnings from https://ethernaut.openzeppelin.com/.

Trying to use [cheb3](https://github.com/YanhuiJessica/cheb3) (web3 CTF tool based on web3.py) as much as possible as CTF practices. The tool is developed and maintained by my teammate at Project Sekai CTF. You can read [here](https://cheb3.readthedocs.io/en/latest/interaction_basis.html) for the documentation.
